# Segue os passos
1. Criar ambiente virtual

   ```sh
   # executar mesmo nivel pasta app
   python -m venv ./venv
   ```

2. Habilitar ambiente virtual

   ```sh
   # win executar mesmo nivel pasta app
   .\venv\Scripts\Activate
   ```

   ```sh
   # linux executar mesmo nivel pasta app
   source venv/bin/activate
   ```
